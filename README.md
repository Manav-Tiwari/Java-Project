CampusCred
A Virtual Credit-Based Campus Marketplace

Overview
As college students, acquiring textbooks, reference materials, and quality notes every semester can be a massive financial burden. Traditional buying and selling is inefficient, and finding someone who wants exactly what you have for a direct trade is nearly impossible.

Enter CampusCred. This command-line Java application serves as a localized, peer-to-peer marketplace where students can trade academic resources using a self-contained virtual economy. Instead of spending real money, users start with a base balance and earn "credits" by selling their old materials, which they can then spend to acquire what they need for the current semester.

Built as a Bring Your Own Project (BYOP) submission, CampusCred leverages core Object-Oriented Programming (OOP) principles, the Java Collections Framework, Exception Handling, and File I/O for data persistence. It provides a lightweight, resilient, and highly practical solution to a genuine, everyday campus problem.
